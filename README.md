# Aplicaciones-Avanzadas-Ecommerce-Guzzi

## Desarrolladores

Para evitar conflictos de integracion, cada desarrollador debe trabajar en su propia rama.

No trabajar directamente en la rama principal.

## Layout del sitio

Cada modulo tiene el mismo layout:

- HEADER
- NAVBAR
- MAIN (Contenido de la pagina)
- FOOTER

### Importante para el desarrollo:

Cada desarrollador deberá modificar únicamente el contenido de los contenedores:

< main > </ main >

Esto evita romper la estructura global del sitio.

También, si es necesario agregar archivos css, js, o imagenes guardenlos en la carpeta /assets. Al momento de referenciar su ruta css, tengan en cuenta que todas las paginas llaman a 'styles.css'.

Pueden agregar su js en < script > sin problema.

#### Tengan cuidado con las rutas ( href="" )
